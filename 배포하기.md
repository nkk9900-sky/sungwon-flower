# 배포하기 (한 번만 보면 됨)

**최종 자료 업데이트:** 어제 회사에서 수정·배포 완료.

**성원플라워 대시보드**는 GitHub에 올리면 **Vercel이 자동으로 배포**합니다.  
별도 명령어 없이 **push만 하면** 됩니다.

---

## 할 일 (수정한 뒤 배포할 때)

**PowerShell**에서 **성원플라워 프로젝트 폴더**로 이동한 뒤 아래 **세 줄**만 실행하세요.

```powershell
cd c:\성원플라워
git add .
git commit -m "수정 내용"
git push
```

- `"수정 내용"`은 아무 메시지나 적어도 됩니다 (예: "매출 표 수정", "2월 23일").
- `git push`까지 하면 GitHub에 올라가고, 곧 **sungwon-flower.vercel.app** 에 반영됩니다.

---

## 정리

| 하고 싶은 일     | 하는 것                    |
|------------------|----------------------------|
| 배포 (수정 반영) | `git add .` → `commit` → `push` |
| 최신 받기        | `git pull`                 |
